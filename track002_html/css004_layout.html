2. 배치

  • float : 왼쪽이나 오른쪽에 옆에 붙이기 (끊기 - clear:both)
  • position : 정확한 자리를 정해서 붙이기(좌표)
  • display : 불록에 줄서기, 속성바꾸기

종류 :static(기본값), relative(박스위 위치가 남는다), absolute(박스 위지 남지 않는다), fixed(브라우저 고정), sticky(스크롤할때 특점지정에 붍기)

부모 : relative(공간확보) / 자식 : (원하는 위치 배치) - 좌표
